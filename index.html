Humus Simulator/Debugger







	

		
	

	

		
Quick Reference

		
Refer to the Humus language overview for more detail.

		
Statements

		

			
Create an Actor

				
CREATE name WITH behavior

			
Send a Message

				
SEND message TO actor

			
Update Behavior

				
BECOME behavior

			
Define Variable(s)

				
DEF pattern AS expression

			
Abort Message-Event Transaction

				
THROW expression

			
Execute Statement Block

				
expression

		

		
Expressions

		

			
Constant

				
TRUE, FALSE, NIL, or ?

			
(Integer) Number

				
digits* or radix#digits*

			
Symbol

				
#name

			
Pairing and Grouping

				
1, 2, 3 is equivalent to (1, (2, 3)), and () is NIL

			
(Lambda) Abstraction

				
\pattern.expression

			
(Function) Application

				
abstraction(expression)

			
Conditional

				
CASE value OF pattern : expression ... END

			
Statement Block

				
[ statement... ]

		

		
Patterns

		

			
Literal

				
TRUE, FALSE, NIL, ?, number, or symbol

			
Variable

				
name or _ to ignore

			
Tuple

				
pattern, pattern ...

			
Value

				
$name or $(expression)

		

		
Built-Ins

		

			
println

				
An actor that prints to the console any message received

			
eq(value, value)

				
Value equality produces a boolean

			
not(boolean)

			and(boolean, boolean)

			or(boolean, boolean)

				
Boolean predicates produce a boolean or ?

			
neg(number)

			add(number, number)

			sub(number, number)

			mul(number, number)

			div(number, number)

				
Arithmetic operations produce a number or ?

			
compare(number, number)

				
Numeric comparison produces -1 if <, 0 if =, 1 if >

			
less(number, number)

			less_equal(number, number)

			greater_equal(number, number)

			greater(number, number)

				
Numeric predicates produce a boolean or ?
